[Wittgensteins "normative" Aussage im Traktatus:]
: Dann hast Du einfach nicht verstanden, was "praeskriptiv" und "normativ"
: bedeuten. Ein Beispiel: Wenn Du "2 + 2" ausrechnen willst, steht kein
: Mathematiker mit der Pistole hinter Dir, der Dich dazu zwingt "= 4"
: hinzuschreiben. Wenn Du aber etwas anderes hinschreibst, hast Du falsch
: gerechnet. Es ist eine normative Regel, dass die Addition von 2 und 2 4
: ergibt.

Besser gesagt: es ist eine normative Regel, dass das Zeichen
fuer das Ergebnis von 2+2 so aussieht  ->   4. Eine Norm ist
letztenendes eine Konvention ueber Benennungen von Sachverhalten.
Aus einer Menge unendlich vieler moeglicher Benennungen wird
einfach eine ausgewaehlt. So in der Theorie. Nun zur Praxis.
Bei Bogackis Aussage liegt der Fall naemlich anders. Es ist,
wenn wir sie denn normativ zu verstehen versuchen, eine Konvention
fuer die Benennung des Inhalts von Aussagen UEBER Sachverhalte,
naemlich dass dieser "leer" sei. "leer" ist aber nun nicht einfach
ein neuerfundener Name, sondern im Zusammenhang mit Aussagen
bereits belegt im Sinne von "blah blah blah". Damit ist offensicht-
lich, dass Bogackis Aussage bereits eine Bewertung ANDERER
zur Wahl stehender Alternativ-Konventionen beinhaltet. Eine rein
normative Festlegung wie "Aussagen ueber Realitaet heissen "Jaksch-
Aussagen" ist das nun wahrlich nicht.
