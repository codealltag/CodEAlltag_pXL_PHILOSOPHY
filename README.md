# CodE Alltag<sub>pXL</sub> PHILOSOPHY

CodE Alltag<sub>pXL</sub> PHILOSOPHY is part of CodE Alltag, a German-language email corpus. It has been pseudonymized and contains emails covering philosophical issues.
