Prozess ... eine Folge von Zuständen in der Zeit, ohne dass eine
             bestimmte Aussage über Art oder Zusammenhang der
             stattfindenden Zustandsänderungen gemacht wird.

Programm ... setzt ein Form von Code voraus, der ausgeführt wird.

              Das Programm setzt etwas Ausführendes voraus.

Bewusstsein ist also sicher nicht ein Programm, ein Code.
Ein Code wird nicht erlebt. Das Bewusstsein ist dynamisch,
ein Programm ist im Verhältnis zum Prozess statisch.

aktuellen Zustand.
