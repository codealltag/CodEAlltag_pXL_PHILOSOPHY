Mit "alles kann ein System sein" meine ich, dass es keine thematischen
Einschränkungen gibt.

Primär wird eine System durch eine Unterscheidung definiert, was im
System und was außerhalb des Systems ist. Diese Unterscheidung kann
auf verschiedene Weisen erfolgen, etwa durch eine Liste von System-
elementen, oder durch eine operative Unterscheidung.

Wenn von einer Liste von Elementen ausgegangen wird, dann wäre
die Frage, ob diese sinnvollerweise als System betrachtet werden
können. Die Bestandteile eines Autos bilden ein technisches System,
Auto und Fahrer könnten als System betrachtet werden.
Mein Autolenkrad bildet mit dem Blumentopf vom Nachbarhaus
kein System.

Man könnte sich jetzt damit beschäftigen, unter welchen Kriterien
eine Menge von Elementen als System betrachtet werden können,
bzw. welche Fehlerquellen man durch Vereinfachungen erzeugt.

Chemisch, physikalisch, kybernetisch und technisch betrachtet gibt
es geschlossene Systeme und offene Systeme. Bei offenen Systemen
findet ein Austausch von Elementen und/oder Energie/Resourcen statt.

Ich nehme an, dass in deiner mathematischen Definitionswelt so etwas
wie ein "offenes System" gar nicht existiert.


System und Zentrum sind nicht identisch.

Ein System kann als Zentrum und damit als Einheit in einem
größeren System auftreten.

Ein Zentrum kann als Teil in einem System, selbst als
System verstanden werden und modelliert werden.

Es hängt also am Abstraktionsgrad (manche würden sagen,
am Aggregationsgrad), welcher Begriff passt.


Aber ich habe doch gar nicht behauptet, dass darin die gleichen
Muster zu erkennen wären.

Ein technisches Muster wäre etwa eine "Energiequelle" oder
ein "Bedienungselement".

Ein Muster in der kindlichen Entwicklung wäre etwa eine "Frage"
oder ein "Lernerlebnis".

Natürlich kann es nicht darum gehen, dass es in der kindlichen
Entwicklung das Muster "Bedienungselement" gibt, oder dass bei
Maschinen das Muster "Lernerlebnis" eine Rolle spielt.

Die Mustertheorie stellt aber einen begrifflichen und gedanklichen
Rahmen zur Verfügung, der in beiden - und vielen anderen Kontexten -
einsetzbar ist.
