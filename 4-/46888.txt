Information ist ein schwer zu fassender Begriff. Ich würde es
bevorzugen, ihn nicht zu definieren, sondern ihn zur grundlegenden
Nomenklatur zu rechnen wie auch den Begriff der Definition.

Keinesfalls ist Information ein Code. Ein Code ordnet Symbolen Bedeutung
zu. Insofern ist ein Code eine formale Sprache.

Ich würde sagen, die Information, die mit einer Nachricht übertragen
wird, ist ihre Bedeutung für den Empfänger. Das hängt interessanterweise
nicht immer mit einem Code zusammen - es kann ein Code vereinbart sein,
aber es kann auch sein, dass z.B. der Zeitpunkt, zu dem eine Nachricht
übertragen wird, mehr Information vermittelt als die kodierte Nachricht
selbst, oder auch der Absender der Information, oder die Darstellung,
falls der Code mehrere Darstellungen für dieselbe Nachricht unterstützt.

Der Sender einer Nachricht muss nicht bewusst Information kodieren, um
welche zu übertragen. Er überträgt immer mehr Information als er
kodiert, oft mehr, als ihm bewusst ist.

Tja, und dann wirds nicht besser: denn die Information liegt dem
Empfänger nun vor. Er kennt sie. Dadurch erweitert sich sein Wissen.
