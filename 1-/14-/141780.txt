Nein, gerade nicht. Mengen, deren Elemente eineindeutig (=umkehrbar
eindeutig) auf die natürlichen Zahlen abbildbar sind, sind gleich
mächtig. Cantor wies nun durch das Diagonalargument nach, dass alle
folgenden mengen (und noch andere) gleichmächtig sind:
natürliche Zahlen; gerade Zahlen; ganze Zahlen (also inkl. den
negativen); rationale Zahlen (=durch Bruch darstellbar) etc.
All diese mengen haben dieselbe Mächtigkeit.

Abzählbar ist jede menge, wenn man sie so ordnen kann, dass jedes
Element in endlicher Zeit bei einer Aneinanderreihung drankommt.
Damit ist dasselbe ausgesagt wie oben: eine Menge ist abzählbar
unendlich, wenn sie auf die natürlichen zahlen (1,2,3,...) umkehrbar
abbildbar ist.
Beispiel: die ganzen Zahlen können so angeordnet werden:
0, 1, -1, 2, -2, 3, -3, 4, -4, ....
Man erkennt, daß jede beliebig große Zahl z.B. 10 hoch 37 plus 64 in
endlicher Zeit drankommt. Oder formaler, ich bilde wie folgt ab:
(Zeile 1 auf Zeile 2)
(1) 0, 1, -1, 2, -2, 3, -3, 4, -4, ....
(2) 1, 2, 3,  4,  5, 6,  7, 8,  9, ...

Dagegen kann man die reellen Zahlen so nicht anordnen, weil man schon
ewig braucht um von 1 auf 2 zu kommen. Von 3 ganz zu schweigen.
