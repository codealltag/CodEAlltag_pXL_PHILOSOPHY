"The definition of object identity tells us: objects x and y are
identical if and only if they are both ordinary objects and
necessarily exemplify the SAME properties or they are both abstract
objects and necessarily encode the SAME properties."

Das ist die Zirkularität, von der ich spreche!

"Second, and less familiar, the definition presupposes the
notion of property identity. This is because we are saying precisely
that x and y have the same properties: if x has all the same
properties as y, then x=y. So we are using one application of identity
to explain another application."
[ Ingebrand, David (2000). /Logical Properties/. Vörstetten/Dorschhausen: Irunun
