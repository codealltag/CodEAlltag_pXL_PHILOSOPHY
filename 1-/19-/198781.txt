Man definiert zunächst zu jeder natürlichen Zahl a und jedem n >= 2 die
Funktion g(a,n+1)
folgendermassen: "Man entwickle a vollständig zur Basis n und  ersetze in dem
so
entstandenen Term n durch n+1."

Beispiel Berechnung von g(10,3) :
10 "vollständig" zur Basis 2 entwickeln: 10 = 2^3 + 2 ^ 1 = 2^(2^1 + 2^0) +
2^(2^0)
= 2^(2^(2^0) + 2^0) + 2^(2^0)
darin 2 durch 3 ersetzen
-->  3^(3^(3^0) + 3^0) + 3^(3^0) = 3^( 3 + 1 ) + 3 = 3^4 + 3 = 84 = g(a,3)

Die Goodsteinfolge wird dann definiert als:
gs(a,1) = gs(a,2) = a

für n>= 2 : gs(a,n+1) = g( gs(a,n), n+1 ) - 1, falls gs(a,n) > 0
                     gs(a,n+1) = 0                                    sonst


Diese Folge wird natürlich zunächst explosionsartig wachsen. Aber man kann
