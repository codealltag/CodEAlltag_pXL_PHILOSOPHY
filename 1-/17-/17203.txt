Herkömmlich:

Eine (formale) Sprache ist eine Teilmenge der Kleenschen Hülle eines
Alphabetes. Unter einem Alphabet versteht man eine endliche Menge von
Zeichen.

Eine Kodierung ist eine injektive Abbildung von Wörtern einer
Quellsprache auf Wörter einer Zielsprache.

Unter einer Dekodierung versteht man die Umkehrung dieser Abbildung; ist
die Kodierung nicht surjektiv, versteht man darunter die Umkehrung der
Kodierung in der mächtigsten Teilmenge der Zielsprache, mit der letztere
bijektiv ist. Wörter aus der Zielsprache der Kodierung, die nicht
abgebildet werden, heissen nicht dekodierbar.

Eine zerstörende Kodierung ist eine surjektive Abbildung von Wörtern
einer Quellsprache auf Wörter einer Zielsprache.

Unter einer Dekodierung einer zerstörenden Kodierung versteht man die
Umkehrung der Kodierung insofern, dass ein Algorithmus zur Auswahl
jeweils eines Wortes aus der Quellsprache für jedes Wort aus der
Zielsprache existiert und ausgewählt ist.

Eine fraktale Dekodierung ist eine Abbildung einer Zielsprache auf eine
unendliche Folge von Ergebnissprachen mit einem
Dekodierungs-Algorithmus, der die Ergebnissprachen als Abbilder eines
iterierenden Funktionensystems so bestimmt, dass diese gegen die
Quellsprache konvergieren oder ihr in einer Metrik in einem Index
nahekommen insofern, dass eine untere Schranke in der Metrik zwischen
einer Ergebnissprache und der Zielsprache unterschritten wird.

Unter einer fraktalen Kodierung versteht man die Auswahl einer
Quellsprache bei einem gegebenen Dekodierungs-Algorithmus zur Erzeugung
von Abbildern eines iterierenden Funktionensystems zu einer gegebenen
Zielsprache gerade so, dass mit diesem Algorithmus die Dekodierung
exisitert.

Zur Beendigung der Sprachverwirrung werde ich hier zukünftig von einem
Code sprechen, wenn es sich um eine Sache handelt, die gedeutet werden
muss, und um eine (De-)Kodierung, wenn ich die rein formalsprachliche
Sache meine.

In dieser Sprechweise handelt es sich bei der Anordnung der Basen in der
DNS vermutlich um eine Kodierung, nicht jedoch um einen Code.
