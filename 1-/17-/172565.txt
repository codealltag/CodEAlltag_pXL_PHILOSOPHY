Bevor ich auf die verschieden Symbolformen des Denkens eingehe (spaeter)  
kurz ein Hinweis auf eine wichtige Vorausetzung Deines Beitrages.

Du setzt voraus, dass einen Einwirkung der Sprache auf das Denken erfolgt.
Also:

Sprache -> Denken

Weiterhin gehst Du spaeter davon aus, dass eine wechselseitige Beziehung  
stattfindet.

Sprache <-> Denken.

Versuch einmal folgenden Ansatz: Es geht eine Auswirkung des menschlichen  
Denkens auf verschiedene Symbolssysteme aus.  Also etwa ueber mehrere  
Stufen:

Denken -> Symbole -> Laute ... -> .. -> Sprache

Ich gebe Beispiele einer moeglichen historischen Entwicklung (feinere  
Details natuerlich asugelassen):

Denken -> Symbole -> Sprache -> Metasprache -> Mathematik -> ...

oder

Denken -> Symbole -> Kunst -> Philosophie -> ...

oder

Denken -> Symbole -> Technik -> Wissenschaft -> ...

Wo koennte hier ein Feedback einsetzen?
Zwischen den Symbolen und dem Denken?
