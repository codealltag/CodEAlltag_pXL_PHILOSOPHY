Oder (um ein komplizierteres physikalisches Beispiel zu nehmen): ein
Phonon (nicht Photon)? Zur Erklaerung: Man stelle sich ein
Kristallgitter vor, und zwar modellmaessig mit Massepunkten an den
Gitterpunkten und Federn dazwischen:

O /\/\/\ O /\/\/\ O /\...
\        \        \
/        /        /
\        \        \
/        /        /
O /\/\/\ O /\/\/\ O /\...
\        \        \
/        /        /
\        \        \
/        /        /
O /\/\/\ O /\/\/\ O /\...
\        \        \
:        :        :

Dieses Gebilde kann man in Schwingungen versetzen, die ueber die
Federn uebertragen werden. Nun lassen sich diese Schwingungen
mathematisch als eine Ueberlagerung von "Elementarschwingungen"
darstellen. Die Gleichungen, denen eine solche "Elementarschwingung"
genuegt, ist strukturell sehr aehnlich den Gleichungen, mit denen man
Teilchen (Elektronen z.B.) beschreibt - man kann z.B. so einer
Elementarschwingung eine Geschwindigkeit zuordnen, oder eine bestimmte
Energie. Deshalb kann man sagen, dass diese "Elementarschwingungen"
gewisse Teilcheneigenschaften besitzen; man nennt sie "Phononen".

Haben diese Phononen nur eine Realitaet in unserem Nervensystem? Im
Unterschied zu der "physikalisch/chemischen Wirklichkeit"? Oder hat
man sich nicht einfach ein Modell von Gitterschwingungen geschaffen,
das mir eine gewisse Strukturierung unserer Wirklichkeit ermoeglicht,
und ein Begriff aus diesem Modell ist eben "Phonon"?

Und ist das mit dem Apfel nicht ebenso? Dass man in einem Lernprozess
einfach die Wirklichkeit strukturiert und mit Begriffen belegt? Und
ein Begriff fuer eine bestimmte Teilstruktur in diesem Modell ist eben
"Apfel". Die Begriffsbildung ist einfach Teil der Modellbildung, und
der Begriff macht auch nur innerhalb des Modells Sinn (so wie im
Phononenbeispiel der Begriff "Phonon" nur sinnvoll ist, wenn ich die
Gitterschwingungen tatsaechlich auf diese Weise in Elementarschwingungen 
zerlege).
